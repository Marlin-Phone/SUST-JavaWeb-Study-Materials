# 期末考试重点

## 本文的考试信息 
2025-2026 学年第一学期 《Web应用开发技术》课程期末考试 陈景霞老师

## 考试范围
总体考是从 Servlet 开始，到 JavaWeb 高级应用（过滤器与跟踪器）结束。基本不直接考HTML、CSS、JavaScript。

## 考试题型与内容分布

### 选择题（15分）
选择题15分，每题1分，共15题
比较杂乱，涵盖各种零碎的知识点，包括但不限于
1. tomcat 目录结构和配置
2. Servlet 生命周期
3. JSP 语法元素
4. 请求转发与响应重定向
5. 上下文对象与会话对象

### 简答题（15分）
简答题15分，每题5分，共3题
1. MVC设计模式指的是什么？是如何实现的？有什么优势？
2. 什么是会话？会话失效的两种情况是什么？
3. DAO设计模式指的是什么？是如何实现的？有什么优势？

### 编程题（70分）
编程题70分，大题数目忘记了，但是分为四个部分

绝大多数为程序填空，按照出题顺序给出，重点为：
1. [实验一](exp01/) HttpServlet Request+Response、请求转发和响应重定向 + [实验四](exp04/) HttpSession API + [实验五](exp05/) JDBC 四个基本步骤（加载驱动、建立连接、执行SQL、关闭资源） 和 改为数据源后的获取连接代码，约25分。
2. [实验三](exp03/) 简单计算器实现(实现JavaBean的创建为直接写Java代码，调用为填空)，约15分。
3. [实验二](exp02/) JSP 每个大的语法都有考（声明、小脚本、表达式、指令、动作、EL表达式），约15分。
4. [实验六](exp06/) 过滤器与监听器，考了过滤器的链式过滤(chain.doFilter(request, response)) 和 监听器忘了，约15分。

以上知识点，除了 JavaBean 的创建为自己写代码外，其他均为填空题。

大题中没有考 xml 的配置。
